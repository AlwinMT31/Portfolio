# Alwin Mathew Tomy - Personal Portfolio

A sleek, responsive, and dynamic personal portfolio website designed to showcase my skills, education, projects, and certifications. Built entirely from scratch using HTML, CSS, and Vanilla JavaScript, this portfolio features smooth animations, a custom dark/gold theme, and a fully functional contact form.

![Portfolio Preview Showcase](alwin.jpg)

## 🚀 Features

- **Modern & Responsive Design:** Built using CSS Flexbox and Grid to ensure perfect rendering across all devices (Desktop, Tablet, Mobile).
- **Custom Theming:** Utilizes CSS Variables (`--bg-dark`, `--accent-gold`, etc.) for consistent styling and easy theme modifications.
- **Dynamic Animations:** Features a sleek preloader and staggered reveal animations in the Hero section using vanilla JavaScript and CSS transitions.
- **Interactive Navigation:** Smooth scrolling navigation links that smoothly anchor to different sections of the page.
- **Working Contact Form:** Integrated with [Web3Forms](https://web3forms.com/) to allow seamless message delivery directly to my email without needing a backend server.

## 📂 Sections

1. **Home:** An engaging landing page with an animated introduction and a quick summary of who I am.
2. **About Me:** A brief overview of my background, interests, and professional aspirations.
3. **Skills:** Categorized showcase of my technical abilities, including:
   - Python, JavaScript, HTML, CSS
   - Machine Learning (Logistic Regression, Random Forest, XGBoost)
   - Natural Language Processing (NLP)
   - Database Management & Data Structures
4. **Training:** Information regarding professional and summer training sessions I've attended.
5. **Education:** Detailed academic timeline from Matriculation and Intermediate up to my B.Tech degree.
6. **Projects:** In-depth views of my most prominent work, including:
   - *Smart Campus Management System*
   - *Credit Card Fraud Detection Platform*
   - *Grievance Redressal Platform*
7. **Certificates:** Rendered visual previews of my completed courses from NPTEL, IBM, and UAB coursera.
8. **Contact:** An active, email-integrated messaging form for quick communication.

## 🛠️ Technologies Used

- **HTML5:** Semantic structuring and sectioning.
- **Vanilla CSS3:** All styling, flex-layouts, SVGs, and keyframe animations.
- **JavaScript (ES6):** Preloader logic and dynamic class toggling for the reveal effects.
- **Web3Forms API:** Serverless email forwarding for the Contact section.

## 💻 Running Locally

Since this is a static site, you don't need any complex build tools to run it. 

### Method 1: Direct File Open
Simply clone or download this repository, and open `index.html` directly in any modern web browser.

### Method 2: Python Local Server
If you prefer to run it through a local HTTP server to prevent CORS issues or cache bugs:
1. Open your terminal in the project directory.
2. Run the following command:
   ```bash
   python -m http.server 8000
   ```
3. Open your browser and navigate to `http://localhost:8000/`.
